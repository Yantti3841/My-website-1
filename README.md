# 陈彦霆 Eric Chen 个人作品网站

这是一个静态个人作品网站，用于展示项目案例、个人能力、服务内容、B 站内容账号和联系方式。

## 预览方式

直接打开 `index.html` 即可预览。

如果希望用本地服务预览，可以在当前文件夹运行：

```bash
python -m http.server 5173
```

然后打开：

```text
http://127.0.0.1:5173
```

## 文件结构

- `index.html`：页面结构
- `styles.css`：视觉风格、响应式布局和动效
- `app.js`：项目数据、作品筛选、详情弹窗和频道卡片
- `site-assets/`：网页优化后的封面图和展示图
